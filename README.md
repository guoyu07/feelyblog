FeelyBlog
=========

FeelyBlog是一个使用PHP+MySQL的轻量级个人博客程序。

由zyzsdy开发。

FeelyBlog is a Lightweight CHINESE PHP blog. 

## 安装方法

1. 请执行根目录下的install.php进行安装设定。

2. 安装结束后请立即删除此文件。
